# OptiML Data Analysis AB — AI/ML & Optimization Case Studies

This repository contains demonstration Jupyter notebooks (simulated case studies) illustrating applied optimization and machine learning approaches.

Each notebook is provided for educational and demonstration purposes only and uses synthetic or publicly available data. They are not based on confidential client work.
